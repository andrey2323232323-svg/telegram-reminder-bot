# Telegram reminder bot

Бот принимает текстовые и голосовые сообщения на русском, вытаскивает дату/время и отправляет напоминание в нужный момент.

## Примеры сообщений

- `завтра в 10:30 позвонить врачу`
- `25 мая в 18:00 оплатить аренду`
- `через 2 часа проверить духовку`
- голосом: `напомни завтра утром отправить документы`

## Команды

- `/start` - краткая справка
- `/list` - список активных напоминаний
- `/cancel 12` - отменить напоминание с ID 12

## Запуск на VPS

1. Создай бота в Telegram через `@BotFather` и получи `TELEGRAM_BOT_TOKEN`.
2. Скопируй `.env.example` в `.env` и заполни токены.
3. Запусти:

```bash
docker compose up -d --build
```

Логи:

```bash
docker compose logs -f reminder-bot
```

Остановка:

```bash
docker compose down
```

## Переменные окружения

- `TELEGRAM_BOT_TOKEN` - токен Telegram-бота.
- `OPENAI_API_KEY` - ключ OpenAI для расшифровки голосовых.
- `OPENAI_TRANSCRIBE_MODEL` - модель для распознавания речи, по умолчанию `whisper-1`.
- `TIMEZONE` - часовой пояс, по умолчанию `Europe/Moscow`.
- `DATABASE_PATH` - путь к SQLite-базе.
